# React Component Editor

A full-stack React component editor with Monaco integration.

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development servers
pnpm run dev
```

- **Frontend**: http://localhost:5173
- **Backend**: http://localhost:3001
